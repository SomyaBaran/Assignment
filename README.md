# CMS Backend

**Course Management System (CMS) Backend** built with **Bun, TypeScript, Express, Prisma, PostgreSQL, Zod, and JWT**.  

Provides APIs for:
- User signup & signin
- Course creation & lectures
- Enrollment & subscription with transactions
- JWT-based authentication
